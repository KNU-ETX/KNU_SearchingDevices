## 프로젝트 소개

### 전기 전자 소자 인식 장치, KNU_Searching Devices

학교 실험실에서 처음 보는 소자를 접했을 때 소자의 정보를 검색하는 시간과 정보 정리에 쓰는 시간을 단축하고자 생성 계획, 앱을 실행하여 소자의 종류(저항, 가변저항, 커패시터, 인덕터, IC칩)를 선택한 후 휴대폰의 후면 카메라로 소자를 촬영하면 소자에 대한 정리된 정보를 알려줌

---

## 개발 환경 및 라이브러리

- HTML5
- JavaScript
- Teachable Machine
